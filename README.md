# 🚀 Genspark プロジェクトハブ

Gensparkでの会話を効率的に管理するための、モダンでクリーンなダッシュボードアプリケーションです。

---

## ✨ 主な機能

### 📁 プロジェクト管理
- **5つのプロジェクトフォルダ**（A～E）をデフォルトで用意
- **色分けラベル**（Purple、Blue、Green、Orange、Pink）で視覚的に識別
- **アコーディオン形式**で展開・折りたたみ可能
- **プロジェクトごとの説明エリア**でメモを保存

### 💬 チャット管理
- **チャットの追加・編集・削除**
- **タイトル・URL・説明メモ**を保存
- **外部リンククリック**でGensparkの会話にアクセス
- **空状態の表示**（チャットがない場合）

### 🗂️ サブフォルダ機能
- **サブフォルダ作成**で階層的に整理
- **ネスト構造**でチャットを管理
- **サブフォルダの展開・折りたたみ**
- **サブフォルダごとのチャット追加**

### 🔄 チャット移動機能
- **プロジェクト間でチャットを移動**
- **サブフォルダ間でチャットを移動**
- **移動先選択モーダル**で直感的に操作
- **プロジェクト ↔ サブフォルダ**の双方向移動に対応

### 🎨 UI/UX
- **ダークモード/ライトモード**切替
- **Gensparkブランドカラー**（紫・青）を使用
- **モダンなアニメーション**とモーダルダイアログ
- **レスポンシブデザイン**（PC・スマホ対応）

### 🔍 便利機能
- **リアルタイム検索**（プロジェクト名・チャットタイトル・説明文）
- **⭐お気に入り機能**でよく使うプロジェクトをマーク
- **最近使用フィルター**で最近アクセスしたプロジェクトを表示
- **新しいチャット開始ボタン**でGenspark公式サイトへリンク

### 💾 データ永続化
- **LocalStorage**に全データを自動保存
- **ブラウザを閉じてもデータを保持**
- **リアルタイム同期**で編集内容を即座に反映

## 🛡️ セキュリティ

### URLベースの保護
- **ランダムなURL**で公開（例：`https://www.genspark.ai/api/page_private?id=xxxxxxxx`）
- **推測不可能なID**により、URLを知らない人はアクセス不可
- **パスワード認証は無効化**（シンプルで十分なセキュリティ）

### 認証機能（オプション）
- `js/auth.js` にパスワード認証機能を実装済み
- デフォルトでは **`enabled: false`** で無効化
- 必要に応じて `AUTH_CONFIG.enabled = true` に変更可能
- 詳細は `AUTH_SETUP.md` を参照

## 🎯 使い方

### 1. プロジェクトを展開
- **プロジェクトカードをクリック**してアコーディオンを開く

### 2. チャットを追加
1. プロジェクトを展開
2. **「➕ チャットを追加」** ボタンをクリック
3. **タイトル・URL・説明（任意）** を入力
4. **「追加」** ボタンでチャットを保存

### 3. サブフォルダを作成（任意）
1. プロジェクトを展開
2. **「➕ サブフォルダを追加」** ボタンをクリック
3. **フォルダ名・説明（任意）** を入力
4. **「作成」** ボタンでサブフォルダを作成
5. サブフォルダ内に同様にチャットを追加可能

### 4. チャットを移動
1. チャットの **「⇄」移動ボタン** をクリック
2. **移動先のプロジェクト**と**サブフォルダ（任意）**を選択
3. **「移動」** ボタンで移動を実行

### 5. 便利機能を活用
- **検索ボックス**でプロジェクト名やチャットタイトルを即座にフィルタリング
- **⭐お気に入り**でプロジェクトをマークし、サイドバーの「お気に入り」から素早くアクセス
- **ダークモード**（右上の 🌙/☀️ アイコン）でテーマ切替
- **プロジェクト説明エリア**に入力すると自動保存

## 🔧 推奨ワークフロー

1. **Gensparkで新規会話を開始** → URLを取得
2. **ダッシュボードを開く** → 該当プロジェクトを展開
3. **「チャットを追加」** → タイトル・URLを入力して保存
4. **次回以降はダッシュボードから過去のチャットを参照**

## 🎨 カスタマイズ

プロジェクト名や色は自由に変更可能です。詳細は **`CUSTOMIZATION.md`** を参照してください。

### 簡単な変更例
- **プロジェクト名**: `index.html` の `<h3>` タグを編集
- **カラー**: `css/style.css` の `:root` 変数を変更
- **プロジェクト数**: `index.html` にカードを追加して `js/app.js` でデータ構造を更新
